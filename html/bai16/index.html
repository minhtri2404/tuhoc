git clone dùng để lấy bài từ githup về máy.

Bước 1: Vào terminal->newTerminal->bấm vào dấu +-> select default profile-> bấm chọn gitBash
Bước 2: Ghi câu lệnh git status vào terminal, câu lệnh này để kiểm tra xem thư mục có git chưa, nếu chưa xuống bước 3.
Bước 3: Ghi câu lệnh git init, câu lệnh này khởi tạo thành công thư mục .git
Bước 4: Quay lại bước 1 sẽ thấy terminal ghi file chưa được theo dõi.
Bước 5: Nếu muốn gửi tất cả thì dùng lệnh: git add . Nếu muốn gửi lẻ thư mục hoặc file thì dùng lệnh: git add thumuc/ hoặc git add add1.html Xong dùng lệnh: git commit -m "first commit" sẽ hiện ra lỗi ko có email và tên. $ git config --global user.email "" lưu ý: email là lúc mình tạo trên githup. $ git config --global user.name "" lưu ý: tên là lúc mình đặt tên trên githup. Xong rồi ghi lại dòng lệnh: git commit -m "first commit" rồi ấn enter.
Bước 6: ghi dòng lệnh git branch -M main, câu lệnh này sẽ chuyển tử master thành main
Bước 7: git remote add origin https:// đẩy lên githup mà dự án mình tạo
Bước 8: git push -u origin main
